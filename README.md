# SpyderDroid

An android controller for [sPyder](https://github.com/j-kowal/sPyder) system 

## Installation


1. ```git clone https://github.com/j-kowal/SpyderDroid.git```

2. Add generated [google-services.json](https://firebase.google.com/docs/android/setup) file to authenticate the app

3. Change firebase credentials [here](https://github.com/j-kowal/SpyderDroid/blob/master/app/src/main/java/com/ncirl/jank/spyderdroid/MainActivity.java#L51)

4. Build and deploy on your phone
